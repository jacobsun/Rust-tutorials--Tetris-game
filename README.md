# Rust Tetris game

![screenshots](https://github.com/jacobsun/Rust-tutorials--Tetris-game/blob/master/screenshots.gif "game sreenshots")

A tetris demo with piston, you may need SDL library to build it.
And you can comment out
```
piston2d-sprite = "0.47.0"
piston-ai_behavior = "0.26.0"
```
in Cargo.toml.
